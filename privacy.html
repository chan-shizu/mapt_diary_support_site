プライバシーポリシー

最終更新日: 2024年8月10日

map diary（以下「本アプリ」）は、ユーザーのプライバシーを最重要視しています。本プライバシーポリシーでは、本アプリがどのような情報を収集し、どのように使用・保護するかについて説明します。

1. 収集する情報

1.1 位置情報

  • 収集目的: 思い出を記録する場所の特定
  • 収集タイミング: ピン作成時、ユーザーが位置情報の使用を許可した場合のみ
  • 保存先: デバイス内およびFirebase Cloud Firestore（暗号化済み）

1.2 写真・画像データ

  • 収集目的: 思い出の記録・保存
  • 収集タイミング: ユーザーが写真を撮影またはライブラリから選択した時
  • 保存先: Firebase Cloud Storage（暗号化済み）

1.3 音声データ

  • 収集目的: 音声メモの記録・保存
  • 収集タイミング: ユーザーが音声録音機能を使用した時
  • 保存先: Firebase Cloud Storage（暗号化済み）

2. 情報の利用目的

  • 思い出の記録・管理・表示機能の提供
  • アプリの機能改善・バグ修正
  • 技術的サポートの提供
  • セキュリティの維持・向上

3. 使用するサードパーティサービス

本アプリは以下のサードパーティサービスを使用しています：

3.1 Firebase (Google LLC)

  • 使用サービス: Cloud Firestore, Cloud Storage, Analytics
  • データ処理地域: 米国・その他のGoogle指定地域
  • プライバシーポリシー: https://policies.google.com/privacy

3.2 Google Maps (Google LLC)

  • 使用目的: 地図表示・位置検索
  • プライバシーポリシー: https://policies.google.com/privacy

4. プライバシーポリシーの変更

本プライバシーポリシーは、法令の変更やサービス内容の変更に応じて更新される場合があります。重要な変更がある場合は、アプリ内通知またはウェブサイトでお知らせします。

5. お問い合わせ

本プライバシーポリシーに関するお問い合わせは、サポートページからご連絡ください。
